# Distributed & Parallel Computing (CPD)

**Course**: Distributed & Parallel Computing  
**Institution**: FEUP - Faculdade de Engenharia da Universidade do Porto  
**Academic Year**: 2024/2025

## Projects

This repository contains the assignments developed for the Distributed & Parallel Computing course:

- **Assignment 1**: Matrix Multiplication Performance Analysis - Analysis of memory hierarchy impact and parallelization on matrix multiplication algorithms in C++ and Haskell (chosen language)
- **Assignment 2**: Distributed Chat System - TCP-based client-server chat application with user authentication, multiple rooms, and AI integration

---

## Assignment 1 - Matrix Multiplication Performance Analysis
Analysis of memory hierarchy access impact on processor performance, comparing different matrix multiplication algorithms with single-core and multi-core implementations using OpenMP.

## Assignment 2 - Distributed Chat System
Implementation of a concurrent client-server chat system in Java with token-based authentication, fault tolerance, and AI-powered rooms using local LLM integration (Ollama).
